# TabBar 图标说明

## 需要的图标文件

请准备以下 4 个图标文件（81x81px，PNG 格式）：

### 首页图标
- `home.png` - 未选中状态（灰色轮廓房子图标）
- `home-active.png` - 选中状态（紫色填充房子图标，#8B5CF6）

### 我的图标
- `profile.png` - 未选中状态（灰色轮廓人像图标）
- `profile-active.png` - 选中状态（紫色填充人像图标，#8B5CF6）

## 图标获取方式

### 方案一：iconfont（推荐）
1. 访问 https://www.iconfont.cn/
2. 搜索 "home" 和 "user"
3. 下载 PNG 格式（81x81px）
4. 使用图片编辑工具调整颜色：
   - 未选中：#4B5563（灰色）
   - 选中：#8B5CF6（紫色）

### 方案二：IconPark
1. 访问 https://iconpark.oceanengine.com/
2. 搜索需要的图标
3. 下载并调整颜色

### 方案三：使用 Emoji（临时方案）
如果暂时没有图标，可以使用纯色方块作为占位：
- 创建 81x81px 的纯色图片
- 未选中：灰色 #4B5563
- 选中：紫色 #8B5CF6

## 图标规范

- **尺寸**: 81x81 像素（推荐）或 40x40 像素（最小）
- **格式**: PNG（支持透明背景）
- **颜色**: 
  - 未选中: #4B5563
  - 选中: #8B5CF6
- **样式**: 简洁、清晰、易识别

## 放置位置

将准备好的图标文件放在当前目录（`src/assets/tabbar/`）下。
