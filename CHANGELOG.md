# 更新日志

## V1.2.0 (2025-12-19) 📎

**多附件支持 & 分享优化**

### 📎 多附件功能
- **多源合并附件**：支持同时上传多个 PDF/图片附件
- **附件原名保留**：上传时保留原始文件名
- **数据库扩展**：events 表新增 attachments JSONB 字段
- **小程序展示**：DetailModal 支持显示多个附件

### 🔗 分享功能重构
- **微信原生分享**：使用 `openType="share"` 触发微信分享菜单
- **分享卡片跳转**：点击分享卡片可直接打开对应活动详情
- **分享图标优化**：换成简约紫色上传箭头图标

### 📅 日历功能优化
- **年月格式支持**：支持"2026年2月"等只有年月的日期格式
- **智能日期解析**：自动使用该月最后一天作为日历事件日期

### 🧹 代码精简
- **移除单源模式**：管理后台只保留多源合并功能
- **移除冗余文字**：小程序不再显示"多源合并(xx个信息源)"

### 🔧 技术改进
- 首页/收藏/历史页统一添加 useShareAppMessage 配置
- 支持 eventId 参数自动打开详情弹窗
- FeedItem 类型添加 attachments 字段

## V1.0.0 (2025-12-12) 🎉

**UniFlow (智汇流)** 正式发布！

### ✨ 核心功能

#### 小程序端
- 📱 信息流展示：活动、招聘、讲座聚合
- 🔍 智能搜索：关键词搜索、分类筛选、过期筛选
- 📋 详情查看：结构化信息、一键添加到日历
- ❤️ 收藏功能：跨设备同步
- 🕐 浏览历史：自动记录
- 🔗 分享功能：一键复制
- 💬 意见反馈：结构化收集

#### 管理后台
- 🤖 AI 智能采集：文本/URL/图片多模态输入
- 🌐 多语言输出：中文/英文/双语
- 📝 内容管理：完整的 CMS 功能
- 📊 数据看板：统计分析

### 🔧 技术栈
- 小程序：Taro 4.x + React 18 + TypeScript
- 管理后台：Next.js 14 + TypeScript
- 后端：Supabase + DeepSeek API
