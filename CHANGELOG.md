# 更新日志

## V1.0.0 (2025-12-12) 🎉

**UniFlow (智汇流)** 正式发布！

### ✨ 核心功能

#### 小程序端
- 📱 信息流展示：活动、招聘、讲座聚合
- 🔍 智能搜索：关键词搜索、分类筛选、过期筛选
- 📋 详情查看：结构化信息、一键添加到日历
- ❤️ 收藏功能：跨设备同步
- 🕐 浏览历史：自动记录
- 🔗 分享功能：一键复制
- 💬 意见反馈：结构化收集

#### 管理后台
- 🤖 AI 智能采集：文本/URL/图片多模态输入
- 🌐 多语言输出：中文/英文/双语
- 📝 内容管理：完整的 CMS 功能
- 📊 数据看板：统计分析

### 🔧 技术栈
- 小程序：Taro 4.x + React 18 + TypeScript
- 管理后台：Next.js 14 + TypeScript
- 后端：Supabase + DeepSeek API
