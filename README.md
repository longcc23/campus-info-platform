# 🚀 UniFlow 智汇流

**UniFlow** 是一台对抗校园信息熵增的 AI 引擎。我们利用多模态 AI 技术，将散落在群聊、海报中的碎片信息，重塑为结构化、中英双语的数据资产。

## 🌟 产品核心功能

### 📱 极简小程序 
- **高效信息流**：按招聘、讲座、活动分类展示，内置一键搜索。
- **隐藏过期**：动态清洗时效，自动过滤已结束的机会。
- **沉浸式详情**：极简排版，清晰展示公司、岗位、截止日期及联系方式。
- **一键锁日程**：智能识别关键时间点，一键同步至系统日历，不错过任何 Deadline。
- **文件直看**：支持 PDF 附件直接在应用内预览。
- **个人资产**：稳定追踪收藏夹与浏览历史，个性化管理头像与昵称。

### 💻 采集管理台 
- **AI 智能采集**：支持文本、URL 链接、海报截图、PDF 文档的快速解析。
- **多源合并 (Multi-Source)**：允许同时导入多份素材（如 JD + 详情网页），由 AI 自动去重合并为单一标准事件。
- **中英双语视野**：AI 自动生成标准化中英双语对照内容，打破信息壁垒。
- **实时看板**：监控平台活跃数据与信息质量。
- **反馈闭环**：直接管理来自用户的意见反馈。

## 🎨 设计理念
- **随手记、随手查、不打扰**：坚持做工具而非社交平台。
- **极简即美**：全站清华紫视觉体系，沉浸式交互体验。
- **数据确权**：所有信息均可追溯原始来源，支持一键 Copy。

## 🚀 快速部署

### 管理后台部署到 Vercel

1. **Fork 本仓库**到你的 GitHub 账号

2. **登录 [Vercel](https://vercel.com)** 并导入项目：
   - 选择 `campus-info-platform` 仓库
   - **Root Directory**: `admin-console`
   - **Framework**: Next.js

3. **配置环境变量**：
   ```
   NEXT_PUBLIC_SUPABASE_URL=你的_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=你的_supabase_anon_key
   DEEPSEEK_API_KEY=你的_deepseek_api_key
   ```

4. **点击 Deploy** 完成部署

详细部署指南请查看 [DEPLOYMENT.md](docs/DEPLOYMENT.md)

### 小程序发布

1. 使用微信开发者工具打开项目根目录
2. 运行 `npm run build:weapp` 构建小程序
3. 在开发者工具中预览和发布
