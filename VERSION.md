## 🎯 核心特性

### 小程序端（Mini-Program）
- 📱 **信息流展示**：活动、招聘、讲座信息聚合展示
- 🔍 **智能搜索**：关键词搜索、分类筛选、过期筛选
- 📋 **详情查看**：结构化信息展示、一键添加到日历
- ❤️ **收藏功能**：保存感兴趣的内容，跨设备同步
- 🕐 **浏览历史**：自动记录浏览记录，支持清空
- 🔗 **分享功能**：一键复制分享内容
- 💬 **意见反馈**：结构化反馈收集
- ⚡ **流畅体验**：Skeleton 骨架屏加载、现代化 UI

### 管理后台（Admin Console）
- 🤖 **AI 智能采集**：支持文本/URL/图片多模态输入
- 🌐 **多语言输出**：中文/英文/中英双语
- 📝 **内容管理**：列表、编辑、发布、删除、置顶
- 📊 **数据看板**：统计分析、热门活动展示
- 🔐 **权限管理**：基于 Supabase Auth 的安全认证
