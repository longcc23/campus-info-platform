# TabBar字体优化说明

## 问题

用户反馈底部TabBar的"首页"、"我的"文字太小，看不清。

## 解决方案

### 方案1：调整颜色（已实施）✅

**限制**：微信小程序原生TabBar无法直接修改字体大小。

**优化措施**：
- ✅ 将未选中颜色从 `#9CA3AF` 调整为 `#4B5563`（更深的灰色）
- ✅ 提升对比度，增强可读性

**配置**：
```typescript
tabBar: {
  color: '#4B5563', // 更深的灰色，提升可读性
  selectedColor: '#8B5CF6', // 清华紫
  // ...
}
```

### 方案2：自定义TabBar（如需更大字体）

如果用户需要更大的字体，需要实现自定义TabBar：

1. **创建自定义TabBar组件**：`src/custom-tab-bar/index.tsx`
2. **在 `app.config.ts` 中启用**：`tabBar: { custom: true }`
3. **在每个TabBar页面中引入组件**

**注意**：自定义TabBar会增加开发复杂度，当前颜色优化已足够提升可读性。

---

## 当前状态

- ✅ TabBar文字颜色已优化（更深，更清晰）
- ✅ 选中色使用清华紫，视觉识别度高
- ⚠️ 字体大小无法直接调整（微信小程序限制）

## 后续建议

如果用户仍然觉得字体太小，可以考虑：
1. 实现自定义TabBar（需要额外开发工作）
2. 使用图标+文字组合（但需要准备图标资源）
