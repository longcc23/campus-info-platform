## 📊 设计系统概览

### 颜色系统

| 类型 | 颜色 | 用途 |
|------|------|------|
| **主色** | `#8B5CF6` (清华紫) | 品牌色、主要操作 |
| **主色渐变** | `135deg, #8B5CF6 → #7C3AED` | 背景、按钮 |
| **辅助色** | 蓝色/橙色/绿色/红色 | 标签、状态 |
| **中性色** | 50-900灰度 | 文字、背景、边框 |

### 字体系统

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| **超大标题** | 48rpx | 700 | 主要标题 |
| **大标题** | 40rpx | 700 | 页面标题 |
| **标题** | 36rpx | 600 | 区块标题 |
| **小标题** | 32rpx | 600 | 卡片标题 |
| **正文** | 28rpx | 400 | 正文内容 |
| **次要信息** | 24rpx | 400 | 辅助信息 |
| **辅助信息** | 20rpx | 400 | 标签、备注 |

### 间距系统

| 级别 | 数值 | 用途 |
|------|------|------|
| **XS** | 8rpx | 微小间距 |
| **SM** | 16rpx | 小间距 |
| **MD** | 24rpx | 中等间距 |
| **LG** | 32rpx | 大间距 |
| **XL** | 48rpx | 超大间距 |
| **2XL** | 64rpx | 巨大间距 |
| **3XL** | 80rpx | 极大间距 |
