# 🤖 AI 解析优化说明

## 📋 优化内容

根据用户反馈"关键信息我觉得没解析完整"，优化了 AI 解析 Prompt，确保提取更完整的信息。

---

## ✅ 已优化的点

### 1. 邮箱地址提取

**问题**：原始素材中的邮箱地址（如 `cdcresume@sem.tsinghua.edu.cn`）没有被提取到"投递链接"字段。

**优化**：
- Prompt 中明确要求提取邮箱地址到 `link` 字段
- 前端"投递链接"字段改为文本输入（不再是 URL only），支持输入邮箱地址
- Label 更新为"投递链接/邮箱 (URL 或邮箱地址)"

### 2. 邮件标题格式提取

**问题**：原始素材中的邮件标题格式要求（如"邮件标题:欧来雅MI:marketing 或电商方向"）没有被提取。

**优化**：
- Prompt 要求如果内容包含邮件标题格式要求，必须在 `summary` 中包含完整信息
- 确保摘要包含完整的投递方式说明

### 3. 摘要详细程度

**问题**：摘要太简短，缺少工作描述的详细信息和投递方式的完整说明。

**优化**：
- 摘要长度从"50字以内"扩展到"100字以内"
- 要求摘要必须包含：
  - 公司名称
  - 岗位方向（包含详细描述概括）
  - 申请群体
  - **完整的投递方式**（包括邮箱地址和邮件标题格式）

### 4. 时间信息提取准确性

**问题**：招聘信息被错误提取了不存在的活动日期和时间。

**优化**：
- 明确区分招聘信息和活动信息
- **招聘信息**：通常没有 `date` 和 `time`，只有 `deadline`
- **活动信息**：有 `date`、`time` 和可能的 `deadline`
- 要求不要将 `deadline` 误提取为 `date` 或 `time`

---

## 📊 提取规则更新

### 链接信息提取

**之前**：
```
link: 只提取 URL 链接
```

**现在**：
```
link: 提取 URL 链接或邮箱地址
- URL: https://example.com
- 邮箱: cdcresume@sem.tsinghua.edu.cn
```

### 摘要要求

**之前**：
```
summary: 50字以内，包含公司名称、岗位类型、关键要求
```

**现在**：
```
summary: 100字以内，必须包含：
- 公司名称
- 岗位方向（详细描述概括）
- 申请群体
- 完整投递方式（邮箱地址 + 邮件标题格式）
```

### 时间信息提取

**规则**：
- **招聘信息**：`date` 和 `time` 应为空字符串，只有 `deadline` 有值
- **活动信息**：`date` 和 `time` 有值，`deadline` 可能有值
